# AI Resume & Interview Preparation Assistant

An AI-powered web application that analyzes resumes, extracts skills, generates personalized interview questions, performs emotion analysis, and provides AI-based feedback with confidence scoring.

## Features
- Resume Upload
- ATS Resume Score
- Skill Detection
- AI Mock Interview
- Emotion Analysis
- Confidence Score
- Downloadable Resume Templates

## Technologies Used
- Python
- Streamlit
- NLP
- TextBlob
- PyPDF2
