🚀 AI Recruitment & Interview Platform

An advanced AI-powered recruitment and career assistance platform built using Python and Streamlit. The system helps candidates improve resumes and interview preparation while helping recruiters analyze applicants efficiently.

✨ Features
📄 ATS Resume Score Prediction
🧠 Resume Skill Extraction
🎯 AI Role Prediction
🎤 AI Mock Interview Questions
😊 Emotion & Sentiment Analysis
⭐ Confidence Score Calculation
📊 Hiring Analytics Dashboard
🏆 Candidate Ranking System
📥 Downloadable Resume Templates
📈 Interactive Charts & Visualizations
